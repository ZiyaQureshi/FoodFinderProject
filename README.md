# Campus Food Finder & Budget Bites

Flutter starter app based on the proposal screens and features:
- Splash screen
- Login / signup
- Home dashboard
- Food finder with search + filters
- Restaurant details
- Budget tracker
- AI meal matcher
- Favorites with reviews

## Run
```bash
flutter pub get
flutter run
```

## Packages used
- sqflite
- path
- shared_preferences
- intl

## Notes
This version uses local seeded sample restaurant data so the app works immediately.
